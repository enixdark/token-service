# TokenService

  * The token service use elixir lang to list all token in blockchain network ( ethereum, bitcoin, etc).

## How to setup
  * Erlang/OTP version 21
  * Elixir version 1.7
  * Phoenix version 1.3.4
  * Mongodb version 4.x.x
  * Update env in config/*.ex

To start your Phoenix server:

  * Install dependencies with `mix deps.get`
  * Start Phoenix server with `mix phx.server`
  * Start Phoenix server with `MIX_ENV=prod mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.

Ready to run in production? Please [check our deployment guides](http://www.phoenixframework.org/docs/deployment).
